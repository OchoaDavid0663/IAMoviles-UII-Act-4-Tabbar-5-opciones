# IAMoviles-UII-Act-4-Tabbar-5-opciones

¡Hola! Oye, ¿me puedes ayudar con un código en Flutter? Quiero hacer una app para 'SIMILARES'. Ya tengo una idea del diseño, quiero que sea oscuro con tonos neón (tipo cian 0xFF00E5FF), pero en lugar de usar un menú de lado (el Drawer), necesito que sea con un TabBar arriba.

El diseño debe quedar así:

Arriba del todo el nombre 'SIMILARES' centrado y con las letras algo separadas.

Abajo del nombre, el TabBar con 5 opciones (puros iconos, nada de texto).

OJO: Para la opción de Souvenirs, no quiero un icono normal, ponle esta imagen como logo: https://raw.githubusercontent.com/OchoaDavid0663/IAMoviles-UII-Act-4-Tabbar-5-opciones/refs/heads/main/logo%20sou.png.

Necesito que el código esté separado. Hazme el main.dart con el controlador de los tabs y luego las 5 páginas aparte en una carpeta que se llame LasPaginas. Cada página solo debe mostrar una imagen central. Estas son las rutas:

medicamentos.dart: https://raw.githubusercontent.com/OchoaDavid0663/IAMoviles-UII-Act-4-Tabbar-5-opciones/refs/heads/main/medicamentos.jfif

souvenirs.dart: https://raw.githubusercontent.com/OchoaDavid0663/IAMoviles-UII-Act-4-Tabbar-5-opciones/refs/heads/main/souvenirs.jfif

suplementos.dart: https://raw.githubusercontent.com/OchoaDavid0663/IAMoviles-UII-Act-4-Tabbar-5-opciones/refs/heads/main/suplementos.png

novedades.dart: https://raw.githubusercontent.com/OchoaDavid0663/IAMoviles-UII-Act-4-Tabbar-5-opciones/refs/heads/main/novedades.jfif

basicos.dart: https://raw.githubusercontent.com/OchoaDavid0663/IAMoviles-UII-Act-4-Tabbar-5-opciones/refs/heads/main/basicos.jfif

Pásame el código completo de los archivos, ¡gracias!
